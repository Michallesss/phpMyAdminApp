# phpMyAdminApp
phpMyAdmin desktop application (need to install phpMyAdmin separately)

"Talk is cheap, show me the code"
~Linus Torvalds

## Features
- responsivity
- remember window position and screen after close and open again
- instant updates
- open source
- macOS, Windows and Linux compatibility

### Shortcuts
```shift+f5``` - reload
```f10``` - open dev tools
